3. Welche Daten erhebt ZUME?

Kurz gesagt: ZUME erfasst keine personenbezogenen Daten.

Alle Funktionen der App arbeiten vollständig lokal auf Ihrem Gerät:

  • Kamera: Wird nur zur Live-Anzeige verwendet. Keine Bilder werden ohne Ihre Zustimmung gespeichert oder übertragen.
  • Fotos: DailyPhoto-Bilder werden ausschließlich in Ihrer lokalen Fotobibliothek gespeichert.
  • Celebrity Match: Die KI-Analyse erfolgt vollständig auf Ihrem Gerät (Core ML). Keine Bilder verlassen Ihr Gerät.
  • OCR & Übersetzung: Die Texterkennung und Übersetzung nutzt iOS-eigene APIs und verarbeitet Daten lokal.

3a. Gesichtserkennung & TrueDepth API

ZUME verwendet die TrueDepth-Kamera und Apple's Vision/ARKit Frameworks für folgende Funktionen:

  • Emotion Coach: Erkennung von Gesichtsausdrücken (Lächeln, Augenbrauen, etc.) zur Stimmungsanalyse
  • Celebrity Match: Vergleich von Gesichtsmerkmalen mit einem lokalen Core ML Modell
  • Spiegel-Modus: Echtzeit-Gesichtserkennung für die Spiegelansicht

Wichtige Datenschutzhinweise zur Gesichtserkennung:

  • Alle Gesichtsdaten werden ausschließlich in Echtzeit verarbeitet
  • Keine Gesichtsdaten werden gespeichert – weder auf dem Gerät noch in der Cloud
  • Keine Gesichtsdaten werden übertragen – die App hat keinen Server-Backend
  • Keine Weitergabe an Dritte – Gesichtsdaten verlassen niemals Ihr Gerät
  • Die Analyse erfolgt Frame für Frame und wird sofort nach der Verarbeitung verworfen

4. Keine Datenübertragung an Dritte

ZUME überträgt keine Daten an externe Server, Analyse-Dienste oder Werbeanbieter. Es gibt:

  • Keine Tracking-Tools (wie Google Analytics oder Firebase)
  • Keine Werbung
  • Keine Social-Media-Integration
  • Keine Cloud-Synchronisation

5. In-App-Käufe

Wenn Sie ZUME Pro erwerben, wird die Transaktion vollständig von Apple über den App Store abgewickelt. Wir erhalten von Apple keine persönlichen Zahlungsdaten.

6. Berechtigungen

ZUME benötigt folgende Berechtigungen:

  • Kamera: Für Spiegel-, Lupen- und DailyPhoto-Modus
  • Fotobibliothek: Zum Speichern und Laden von DailyPhoto-Bildern
  • Benachrichtigungen (optional): Für tägliche DailyPhoto-Erinnerungen

Sie können diese Berechtigungen jederzeit in den iOS-Einstellungen widerrufen.

7. Datensicherheit

Da alle Daten lokal auf Ihrem Gerät verbleiben, profitieren Sie von den Sicherheitsmechanismen Ihres iOS-Geräts, einschließlich Verschlüsselung und Face ID/Touch ID.

8. Ihre Rechte

Da wir keine personenbezogenen Daten erheben, gibt es keine bei uns gespeicherten Daten, auf die Sie zugreifen, die Sie korrigieren oder löschen müssten. Alle Daten liegen ausschließlich auf Ihrem Gerät.

Privacy Policy (English)

Summary

ZUME does not collect any personal data. All features work completely offline and locally on your device. No images, no tracking, no data transfer to servers.

What data does ZUME access?

  • Camera: Used only for live preview. No images are stored or transmitted without your consent.
  • Photos: DailyPhoto images are stored exclusively in your local photo library.
  • Celebrity Match: AI analysis runs entirely on your device using Core ML.
  • OCR & Translation: Uses iOS native APIs, processed locally.

Face Detection & TrueDepth API

ZUME uses the TrueDepth camera and Apple's Vision/ARKit frameworks for the following features:

  • Emotion Coach: Detects facial expressions (smile, eyebrows, etc.) for mood analysis
  • Celebrity Match: Compares facial features with a local Core ML model
  • Mirror Mode: Real-time face detection for the mirror view

Important privacy information regarding face detection:

  • All facial data is processed in real-time only
  • No facial data is stored – neither on the device nor in the cloud
  • No facial data is transmitted – the app has no server backend
  • No sharing with third parties – facial data never leaves your device
  • Analysis is performed frame by frame and immediately discarded after processing

No third-party data sharing

ZUME contains no analytics, no advertising, no social media integration, and no cloud sync.
